# kp9-golang-learn

repo pembelajaran go

## Tugas pertemuan 2

### Cara Menggunakan

1. Pastikan Anda memiliki Go (Golang) terinstal di komputer.
2. Buka terminal atau command prompt.
3. Masuk ke direktori yang ada file `main.go`.
4. Jalankan program dengan perintah berikut:

   ```bash
   go run main.go
   ```
